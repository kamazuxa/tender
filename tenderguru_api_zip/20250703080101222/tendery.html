API для получения информации о тендерах России 

GET запросы к странице

https://www.tenderguru.ru/api/export

Данный раздел API предоставляет подробную информацию о тендерах России, включая государственные (подпадающие под действие 44-ФЗ, 223 ФЗ, 615 ФЗ) и коммерческие тендеры. Данные поступают со всех электронных торговых площадок, а также сайтов фирм, проводящих тендеры.

API предназначен для упрощения работы с данными о тендерах, обеспечивая единый и структурированный доступ к информации с различных площадок и сайтов.
Примеры запроса: https://www.tenderguru.ru/api2.3/export?kwords=%D1%80%D0%B5%D0%BC%D0%BE%D0%BD%D1%82


Основные параметры запроса

kwords – поисковое словосочетание
r{номер}=1 — регион закупки (например Московская область, r50=1).
Пример запроса с фильтрацией по регионам:
https://www.tenderguru.ru/api2.3/export?r33=1&r76=1&kwords=%D1%80%D0%B5%D0%BC%D0%BE%D0%BD%D1%82
Номера регионов: https://www.tenderguru.ru/api2.3/export?mode=regions

Пример запроса с фильтрацией по городу:

https://www.tenderguru.ru/api2.3/export?kwords=%D1%80%D0%B5%D0%BC%D0%BE%D0%BD%D1%82&city_id=34
Номера городов: https://www.tenderguru.ru/api2.3/export?mode=cities

Разделы:

c{номер}=1 — раздел закупки (например, Строительство, r1=1).
Список разделов по тендерам:
https://www.tenderguru.ru/api2.3/export?mode=cat

44ФЗ, 223ФЗ, коммерческие

Параметр - f

  • f = 44 (44-ФЗ)
  • f = 223 (223-ФЗ)
  • f = kom (коммерческие)

Либо передачей параметра kom, как в старой версии API:
kom=1 – 44 - ФЗ, kom=3 — 223 - ФЗ, kom=2 - коммерческие

Примеры:
https://www.tenderguru.ru/api2.3/export?f=44&kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87
https://www.tenderguru.ru/api2.3/export?f=223&kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87
https://www.tenderguru.ru/api2.3/export?f=kom&kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87

Площадки

e{номер}=1 — площадка (например, Сбербанк-АСТ, e1=1).
Информация о площадке: https://www.tenderguru.ru/api2.3/etp/{номер площадки}
https://www.tenderguru.ru/api2.3/etp/1
Список площадок по тендерам:
https://www.tenderguru.ru/api2.3/export?mode=eauc Пример запроса по тендерам одной площадки:
https://www.tenderguru.ru/api2.3/export?kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87&e1=1
