Раздел API о иностранных закупках 

GET запросы к странице

https://www.tenderguru.ru/api2.3/export/world

API обеспечивает доступ к базе данных иностранных закупок со всего мира. Позволяет просматривать актуальные тендеры, сортировать и фильтровать их по различным параметрам, таким как страна, дата и т.д.



					
	
						
						
Выборка зарубежных тендеров

						
GET запросы должны идти к странице:

						https://www.tenderguru.ru/api2.3/export/world
						

						
											

					
										

					

					
Поля в запросе

						

							
							

								Поле
								Тип
								Описание
							

							
							
							
									

										kwords
										string
										Ключевые слова для поиска
									

									
									

										api_code
										string
										Токен пользователя
									

									
									

										dtype
										string
										xml по умолчанию, json
									

									
									

										kwords_minus
										string
										Ключевые слова для исключения из поиска
									

									
									

										irazd
										int
										Режим поиска (везде - 0, в заговках - 1, по номеру - 3)
									

									
									

										page
										int
										Номер страницы
									

									
									

										price1
										int
										Минимальная цена
									

									
									

										price2
										int
										Максимальная цена
									

									
									

										date_start
										string
										Дата начала публикации тендера
									

									
									

										date_end
										string
										Дата окончания публикации тендера
									

									
									

										sort_by
										string
										Параметр для сортировки результатов
По дате - by_date (по умолчанию)
По кол-ву дней до конца подачи заявок - by_date_end
По стоимости - by_price		
									

									
									

										sort_dest
										string
										Направление сортировки
desc по убыванию
asc по возрастанию		
									

									
									

										r{номер страны}
										
										r{номер}=1 — страна закупки (например, Казахстан, r113=1 - справочник стран)
									

									
									

										e{номер площадки}
										int
										e{номер}=1 — площадка (например, Государственные закупки Китая, e515=1)
