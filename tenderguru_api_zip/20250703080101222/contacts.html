API для получения информации о контактах организаций. 

GET запросы к странице

https://www.tenderguru.ru/api2.3/export/contragent/inn/{ИНН}/contact

Все контакты юридических лиц взятые из открытых источников. Входят e-mail адреса, телефоны и контактные лица. Полная информация только для подписчиков портала. Примеры запроса:

https://www.tenderguru.ru/api2.3/export/contragent/inn/8620001915/contact
