API информации о контрагенте по ИНН 

GET запросы к странице

https://www.tenderguru.ru/api2.3/export/contragent/inn/{ИНН организации или ИП}

API предназначен для обеспечения программного доступа к базе данных организаций и индвидуальных предпринимателей и их информации. Он позволяет получить обширные данные о предприятиях, исходя из ИНН или ключевых слов, связанных с видом деятельности организации.

Например:
https://www.tenderguru.ru/api2.3/export/contragent/inn/5003093330
https://www.tenderguru.ru/api2.3/export/contragent/inn/5003093330?dtype=json (в json)

Информация контрагентах по тематике

По отраслям возможно посмотреть по ключевому слову или по кодам ОКВЭД2:

Примеры запроса контрагетов по тематике "строительство":

https://www.tenderguru.ru/api2.3/export/contragent?kwords_tender=%D1%81%D1%82%D1%80%D0%BE%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D1%82%D0%B2%D0%BE

Примеры запроса контрагентов по тематике 46.42.1 - Торговля оптовая одеждой:

https://www.tenderguru.ru/api2.3/export/contragent?d1=46&d2=42&d3=1

Информация о поставляемой продукции организации

API предоставляет детальную информацию о продукции, которую контрагент предоставляет на рынок. Это включает в себя характеристики товаров, их стоимость, наличие и другие параметры.
https://www.tenderguru.ru/api2.3/export/contragent/inn/5003093330?mode

Информация о заказчиках контрагента

Ответ:

Customer - Название организации заказчика
INN - ИНН
ContractSum – сумма контрактов, рубли
KolvoKontracts  — кол-во контрактов
ApiLink — ссылка на заказчика

С помощью этого API можно получить список заказчиков контрагента, а также детали сотрудничества, включая объемы поставок, частоту заказов и отзывы о качестве продукции или услуг (параметр mode=customers).
https://www.tenderguru.ru/api2.3/export?mode=customers&inn=7716615618

Информация о поставщиках контрагента

Ответ:

Org — Название организации поставщика
INN - ИНН
SumContracts – сумма контрактов, рубли
KolvoContracts — кол-во контрактов
ApiLink — ссылка на поставщика

API позволяет изучить список поставщиков, с которыми контрагент сотрудничает, узнать условия партнерства, а также получить информацию о товарах или услугах, которые контрагент закупает у своих поставщиков (параметр mode=suppliers).
https://www.tenderguru.ru/api2.3/export?mode=suppliers&inn=9204576208

Информация о документах контрагента

Этот API предоставляет доступ к документам контрагента, включая договоры, лицензии, сертификаты и другие официальные документы, подтверждающие легальность деятельности и соответствие стандартам.
https://www.tenderguru.ru/api2.3/export/contragent/inn/5003093330

Информация о контактах организации

Выгрузка контактной информации организации. Включает в себя телефоны, адреса, адреса электронной почты и другую актуальную контактную информацию.
https://www.tenderguru.ru/api2.3/export?mode=contact&inn=5003093330&api_code=&dtype=json

Арбитражные дела

Ответ:
date_publ - дата
link — ссылка на kad.arbitr
plaintiff - истец
defendant - ответчик
third_person — третье лицо
another_person — участник процесса
num — номер дела
sum — сумма иска
org_inn - ИНН

Получение данных о наличии арбитражных дел, в которых участвует данная организация. Включает детали о характере спора, участниках, решениях суда и текущем статусе дела.

параметр mode=kad (inn — ИНН заказчика). Пример: https://www.tenderguru.ru/api2.3/export?mode=kad&inn=6671459435
