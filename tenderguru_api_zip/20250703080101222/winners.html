Победители закупок 

GET запросы к странице

https://www.tenderguru.ru/api2.3/export/contracts

Данный раздел API предоставляет подробную информацию о победителях закупок по законам 44 и 223. А так же прогнозы победителей.

Карточка контракта

						
GET запросы должны идти к странице:

						https://www.tenderguru.ru/api2.3/export/contracts
						

						
						

						
	
Данный раздел API предоставляет подробную информацию о контрактах.

	
Если нужно получить продукцию или услуги поставляемую по контракту, то запрос идет к https://www.tenderguru.ru/api2.3/export/contracts/products


Пример:
https://www.tenderguru.ru/api2.3/export/contracts?kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87

https://www.tenderguru.ru/api2.3/export/contracts/products?kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87
В json

https://www.tenderguru.ru/api2.3/export/contracts?kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87&dtype=json

https://www.tenderguru.ru/api2.3/export/contracts/products?kwords=%D0%BA%D0%B8%D1%80%D0%BF%D0%B8%D1%87&dtype=json


Пример карточки контракта

Контакты контрагентов

						
GET запросы должны идти к странице:

						https://www.tenderguru.ru/api2.3/export/contragent/inn/{ИНН}/contact
						

						
						

						

Все контакты юридических лиц взятые из открытых источников. Входят e-mail адреса, телефоны и контактные лица. Полная информация только для подписчиков портала.

Примеры запроса:



https://www.tenderguru.ru/api2.3/export/contragent/inn/8620001915/contact
