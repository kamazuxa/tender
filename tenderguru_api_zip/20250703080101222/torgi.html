Список торгов на реализацию и банкроства

GET запросы должны идти к странице:
https://www.tenderguru.ru/api2.3/export/torgi


Поля в запросе

Поле Тип Описание
kwords string Ключевые слова для поиска
api_code Токен пользователя
dtype string xml по умолчанию, json
kwords_minus string Ключевые слова для исключения из поиска
irazd int Режим поиска
mode string Область поиска
page int Номер страницы
price1 int Минимальная цена
price2 int Максимальная цена
date_start string Дата начала публикации тендера
date_end string Дата окончания публикации тендера
sort_by string Параметр для сортировки результатов
По дате - by_date (по умолчанию)
По кол-ву дней до конца подачи заявок - by_date_end
По стоимости - by_price
org_inn ИНН заказчика
sort_dest string Направление сортировки
desc по убыванию
asc по возрастанию
actual int Вывод тендеров с датой до конца подачи заявок
day int Кол-во дней до конца подачи заявок
r{номер региона} int r{номер}=1 — регион закупки (например Московская область, r50=1)
e{номер площадки} int e{номер}=1 — площадка по торгам (например, Сбербанк-АСТ, e1=1)
c{номер раздела} int c{номер}=1 — раздел торгов (например, Строительство, c1=1)
city_id int Город

Поля в ответе

Поле Тип Описание
Date string Дата торга
Number int Номер торга
TorgiName string Название формы торга
Customer string Заказчик
Category string Категория торга
Region string Регион проведения торга
Price int Начальная цена лота
EndTime string Дата окончания подачи заявок
ETP string Электронная торговая площадка
TorgInnerLink string Внутренняя ссылка на торги
TorgiLink string Ссылка на торги
ApiTorgiLink string API ссылка на торги
ApiEtpTorgiLink string API ссылка на торги площадки
ApiCatTorgiLink string API ссылка на торги по разделу
ApiRegionTorgiLink string API ссылка на торги региона
