Документация API для получения информации о планируемых закупках по законам 44-ФЗ и 223-ФЗ 

GET запросы к странице

https://www.tenderguru.ru/api2.3/export/planzakup

Представление доступа к информации о будущих закупках различных организаций. Благодаря этому, пользователи могут заранее подготовиться к предстоящим тендерам, изучить условия и требования, а также оценить потенциальные возможности для своего бизнеса.

Пример: https://www.tenderguru.ru/api2.3/export/planzakup?kwords=%D1%80%D0%B5%D0%BC%D0%BE%D0%BD%D1%82
https://www.tenderguru.ru/api2.3/export/planzakup?kwords=%D1%80%D0%B5%D0%BC%D0%BE%D0%BD%D1%82&dtype=json

Для вывода всех планов закупок по заказчику ИНН передается в параметр org_inn или в kwords:

https://www.tenderguru.ru/api2.3/export/planzakup?kwords=105066680
или https://www.tenderguru.ru/api2.3/export/planzakup?org_inn=105066680

Информация по выбранному плану закупок:

https://www.tenderguru.ru/api2.3/export/plans?id=977201975
https://www.tenderguru.ru/api2.3/export/plans?reestr_number=202308761000002001
https://www.tenderguru.ru/api2.3/export/plans?reestr_number=202308761000002001&lot_number=202308761000002001000002

Информация о планируемой закупке

						
GET запросы должны идти к странице:

						https://www.tenderguru.ru/api2.3/export/plans
