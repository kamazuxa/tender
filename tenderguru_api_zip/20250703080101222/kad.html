API для получения арбитражных и судебных дел по ИНН контрагента 

GET запросы к странице

https://www.tenderguru.ru/api2.3/export/contragent/inn/{ИНН организации}?kad=1

Страница API для получения арбитражных и судебных дел по ИНН организации представляет собой мощный инструмент для автоматизированного доступа к обширной базе данных судебных дел и арбитражных процессов. Пользователи могут запросить информацию о судебных делах, связанных с конкретной организацией, используя её ИНН как уникальный идентификатор. API предоставляет детальную информацию о каждом деле, включая номер дела, дату его регистрации, статус, участников. В данный момент не обновляется.
